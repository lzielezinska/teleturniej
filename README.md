# Quiz App

## Requirements

* Visual Studio 2019
* .NET Core 3.0
* SQL Lite

## How to run

1. Create database by using

```bash
dotnet ef database update
```

2. Or update database by passing in Package Manager Console

```bash
Update-Database
```

3. Run project

```bash
dotnet run
```

## Tests

1. To run tests type

```bash
dotnet test
```
